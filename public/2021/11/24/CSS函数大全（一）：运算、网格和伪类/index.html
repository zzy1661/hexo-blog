CSS函数大全（一）：运算、网格和伪类

前端笔记
5k 词

这是我参与11月更文挑战的第24天，活动详情查看：2021最后一次更文挑战

本系列目标是做一个css函数完全指南，将尽可能罗列css中所有的函数及其用法，当需要使用或者理解某个css函数的时候，只要来这里寻找就可以了。

数学运算

calc

最常用的函数之一，可以执行一些计算

但是很多人都不知道的是，这里不仅可以传长度，还可以计算角度、时间，它支持的值可以是

  • <length>、
  • <frequency>(这个单位目前还没有被任何css属性用到), 
  • <angle>、
  • <time>、
  • <percentage>、
  • <number>、
  • <integer>

另外还有一点：**+ 和 - 运算符的两边必须要有空白字符。**，而* 和 / 这两个运算符前后不需要空白字符。

max和min

取最大值或最小值

clamp

当默认值小于最小值时取最小值，大于最大值时取最大值，否则取默认值。它接收三个参数：最小值、默认值、最大值。

clamp() 支持的值有 <length>、<frequency>、<angle>、<time>、<percentage>、<number>、<integer> 

clamp(MIN, VAL, MAX) 其实就是表示 max(MIN, min(VAL, MAX))

网格

除了上面两个涉及到计算的方法，在网格布局中还有两个minmax和fit-content

minmax

设置网格item宽的最大最小值

在网格布局中，max-content代表最小值由内容决定（内容不被拥挤到换行），1fr指最大值占用剩余空间

fit-content

限制设置网格item宽度，当内容不超过这个值时，宽度为内容的宽度，如果内容过多，则宽度最大到这个限制的值

repeat

网格布局中还有一个函数repeat,允许以更紧凑的形式写入大量显示重复模式的列或行。

可以用在grid-template-columns 和 grid-template-rows中。

伪类选择器函数

:nth-child

:nth-last-child

从兄弟节点中从后往前匹配处于某些位置的元素

:nth-of-type

针对具有一组兄弟节点的标签, 用 n 来筛选出在一组兄弟节点的位置。

:nth-last-of-type

:nth-last-of-type(an+b)  这个 CSS 伪类 匹配那些在它之后有 an+b-1 个相同类型兄弟节点的元素，其中 n 为正值或零值。它基本上和 :nth-of-type 一样，只是它从结尾处反序计数，而不是从开头处。

这个系列的选择器应该都很熟了，可以参考下面的例子

:lang

基于元素语言来匹配页面元素。国际化可能用到。

:not

用来匹配不符合一组选择器的元素。由于它的作用是防止特定的元素被选中，它也被称为_反选伪类_（_negation pseudo-class_）。

这个api有两个地方需要注意:

  • 这个选择器只会应用在一个元素上，无法用它来排除所有父元素。比如， body :not(table) a 依旧会应用到表格元素 <table> 内部的 <a> 上, 因为 <tr>将会被 :not(table) 这部分选择器匹配。
  • 由于伪元素不是简单的选择器，他们不能被当作 :not() 中的参数，形如 :not(p::before) 这样的选择器将不会工作。

:is和:where

:matches()和:any()都被重命名为了:is() is函数以一个选择器列表作为其参数，并选择能够被在该列表中的任意一个选择器匹配的元素。

需要注意的是，选择器列表并不支持伪类。 :where()的优先级总是为0，但是:is()的优先级是由它的选择器列表中的优先级决定的。
